# Tris 

Gioco del Tris a due giocatori realizzato in Python, giocabile da terminale.

## Come eseguirlo
Assicurati di avere Python 3 installato.
Vai nel prompt dei comandi e inserisci

Regole

I giocatori si alternano (X e O)

Vince chi completa una riga, colonna o diagonale

Se la griglia è piena → pareggio
